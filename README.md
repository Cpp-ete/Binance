# Binance Futures Testnet Trading Bot

A small Python application that places MARKET and LIMIT orders on Binance USDT-M Futures Testnet using a clean, reusable structure, with logging and error handling.

## Features

- Place MARKET and LIMIT orders on Binance USDT-M Futures Testnet.
- Support BUY and SELL sides.
- CLI input with validation (symbol, side, order type, quantity, price).
- Structured code (client, orders, validators, logging).
- Logging of API requests, responses, and errors to log files.

## Requirements

- Python 3.10+ recommended
- Binance USDT-M Futures Testnet account
- API key and secret for the testnet
